1. 课前准备¶

后四节课以python语言为例，主要帮助同学快速掌握编程的核心逻辑和思维方式。并利用python语言开发属于自己的软件（由于课时量有限，实现如下的一个：翻译软件、语音软件、文献自动下载软件、国家基金信息查询），并搭建一个神经网络和代谢通量模型。

为了保障学习效果，希望每个同学提前安装好所需要的软件。课前准备这一节，同学请按照说明书，配置好需要的软件。

、考虑到，很多同学以后的工作分子生物学实验。所以，第一节我们将补充snapgene的使用方法，教会学生如何精细的推导分子生物学的各项实验过程，包括PCR、重叠PCR、同源重组、Blast、基因标注等基本过程。让学生从具体的操作中，重新理解分子生物学的研究过程。
